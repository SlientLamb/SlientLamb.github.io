Q&A WSL2使用Kex窗口模式 报错连接失败

Q：报错如下

使用窗口模式时kex --win

Error connecting to the KeX server.
Please try "kex start" to start the service.
If the server fails to start, please try "kex kill" or restart your WSL2 session and try again.

A：

重新挂载/tmp/.X11-unixsudo mount -o remount,rw /tmp/.X11-unix-o remount：将一个已经挂下的档案系统重新用不同的方式挂上。例如原先是唯读的系统，现在用可读写的模式重新挂上。-o rw：用可读写模式挂上。

然后vncserverkex
