socks

在python环境中已经安装了socks，但是运行代码还是报错：No module named ‘socks’

解决方法：使用前先更新requests 版本为支持socks的版本。

pip install -U requests[socks]

./pip.exe install -U requests[socks] -i [http://mirrors.aliyun.com/pypi/simple/](http://mirrors.aliyun.com/pypi/simple/) --trusted-host mirrors.aliyun.com
